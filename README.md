# California House Price Prediction

## Overview
This project predicts California house prices using Machine Learning and the California Housing Dataset from Scikit-Learn.

## Models Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Gradient Boosting Regressor

## Final Model Performance
- Train R² Score: 0.8049
- Test R² Score: 0.7756

## Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- GitHub

## Features
- EDA
- Model Comparison
- Feature Importance Analysis
- House Price Prediction Web App
